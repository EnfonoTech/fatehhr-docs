# Install the app

Fateh HR comes in two flavours. Pick whichever fits.

## Android APK (recommended)

The native Android app is the best experience — faster, works fully offline, shows native notifications, and supports fingerprint / face unlock.

### Step 1 — Download the APK

Open this link on your Android phone and tap the latest `fatehhr-demo-<version>.apk`:

**[Latest APK release →](https://github.com/EnfonoTech/fatehhr/releases/tag/frontend-dev)**

### Step 2 — Allow install from your browser

Android asks for permission the first time you install from outside the Play Store.

1. When you tap the APK, Android says **"For your security, your phone is not allowed to install unknown apps from this source."**
2. Tap **Settings** in that popup.
3. Turn on **Allow from this source**.
4. Go back and tap **Install**.

### Step 3 — Open it

After install, tap **Open** (or find "Fateh HR Demo" in your app drawer). You're ready to log in.

::: tip Updating later
When a new version is available your HR team will share the download link again. Tap the new APK → Install → Done. Your login and PIN are preserved across updates.
:::

## Web app (any phone, any browser)

If you can't install the APK, or you're on iPhone, you can use the web version.

### Step 1 — Open the site

Visit: **[https://hr-demo.enfonoerp.com/fatehhr](https://hr-demo.enfonoerp.com/fatehhr)**

(Your company may have its own URL — check with HR.)

### Step 2 — Add to home screen

To make it feel like an app:

**On Chrome / Android:**
1. Open the site
2. Tap the ⋮ menu → **"Install app"** or **"Add to Home screen"**
3. Confirm — an icon appears on your home screen

**On Safari / iPhone:**
1. Open the site
2. Tap the Share button (box with arrow)
3. Scroll down → **"Add to Home Screen"**
4. Name it "Fateh HR" → **Add**

The web version has almost all features of the app, with these differences:
- Camera/selfie works but requires granting browser permission each session on some devices
- No native biometric unlock (PIN only)
- No background sync when the browser tab is closed

## Which one should I use?

| Situation | Use this |
|---|---|
| Android phone, most of the time | **APK** |
| iPhone | **Web app** (no APK for iOS yet) |
| Shared / demo device | Web app |
| Just testing quickly | Web app |
